# 📚 FlaskBookHub

FlaskBookHub is a web-based book management and ordering system built using Flask. It allows users to register, log in, browse books, and place orders easily.

## 🚀 Features
- User Registration & Login
- Secure Authentication
- Browse Books
- Order Books


## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- MySQL
- Git & GitHub

## ▶️ How to Run
1. Clone repository
2. Create virtual environment
3. Install requirements:
   pip install -r requirements.txt
4. Run app:
   python app.py
