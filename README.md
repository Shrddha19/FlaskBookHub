📚 FlaskBookHub

FlaskBookHub is a web-based book management and ordering system built using Flask. It allows users to register, log in, browse books, and place orders easily.

🚀 Features  
👤 User Registration & Login  
🔐 Secure Authentication  
📖 Browse Books  
🛒 Order Books  

🛠️ Tech Stack  
💻 Python (Flask)  
🎨 HTML, CSS  
🗄️ MySQL  
🔧 Git & GitHub  

▶️ How to Run  
📥 Clone repository  
⚙️ Create virtual environment  
📦 Install requirements: pip install -r requirements.txt  
▶️ Run app: python app.py
